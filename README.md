# Scanner Mode v18

Что изменено:
- ручной сканер сохранён как основной сценарий
- возвращён ручной поиск по имени, фамилии и коду
- поиск по коду ищет сразу по коду
- поиск по имени / фамилии:
  - один результат — сразу карточка
  - несколько результатов — список выбора
- сохранены звук "пик", блокировка двойных сканов и папка `cards/`

- поправлена адаптивная верстка формы поиска и загрузки Excel
